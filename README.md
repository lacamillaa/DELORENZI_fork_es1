# DELORENZI_fork_es1
primo progetto con multiprocessing
### consegna:
Il programma deve chiedere all'utente un numero intero positivo N, che rappresenta il numero di processi da creare.
Il programma deve creare esattamente N processi utilizzando la funzione fork().

Ogni processo figlio deve stampare:
Il proprio PID.
Il PID del processo padre (PPID).
Il numero progressivo del processo (es. "Processo figlio 1", "Processo figlio 2", ecc.).
